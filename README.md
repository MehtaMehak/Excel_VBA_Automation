## ğŸ§  Project Descriptions

### 1. `merge_csv_files.bas`
Automatically merges all CSV files from a selected folder into one consolidated Excel sheet.  
**Use Case:** Perfect for combining monthly sales data, financial statements, or exports from multiple systems.  
**Key Features:**  
- Loops through all CSV files in a folder.  
- Appends data dynamically into a master sheet.  
- Automatically removes duplicates and cleans formatting.

---

### 2. `highlight_duplicates.bas`
Highlights duplicate entries in a specified column using conditional formatting.  
**Use Case:** Helps detect repeated entries in ID lists, invoices, or datasets.  
**Key Features:**  
- Works dynamically based on user-defined range.  
- Customizable color codes.  
- Supports large datasets efficiently.

---

### 3. `send_email_report.bas`
Sends an automated email through Microsoft Outlook with the current Excel report attached.  
**Use Case:** Ideal for scheduled reporting or daily/weekly team updates.  
**Key Features:**  
- Integrates directly with Outlook.  
- Automatically attaches the latest file.  
- Allows custom subject lines, recipients, and email body text.

---

### 4. `create_pivot_table.bas`
Creates and refreshes Pivot Tables automatically from a dataset.  
**Use Case:** Saves time by automating monthly or weekly pivot-based reporting.  
**Key Features:**  
- Automatically defines data ranges.  
- Adds multiple fields dynamically.  
- Refreshes pivot tables in one click.

---

### 5. `automate_formatting.bas`
Applies consistent formatting to reports â€” adjusting column widths, applying borders, setting headers, and aligning data.  
**Use Case:** Ensures professional and consistent reporting formats for presentations or client deliverables.  
**Key Features:**  
- Customizable header formatting.  
- Adjusts all columns to fit content.  
- Can be run on multiple sheets at once.
